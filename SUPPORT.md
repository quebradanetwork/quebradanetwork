# Suporte

Precisa de ajuda com um projeto da **Quebrada Network**? Use o canal correto
para agilizar o atendimento.

## 🧭 Onde buscar ajuda

| Situação | Canal |
|----------|-------|
| 🐞 Encontrou um bug | Abra uma [issue](../../issues) com o template de bug |
| ✨ Tem uma sugestão | Abra uma [issue](../../issues) com o template de funcionalidade |
| 💬 Dúvida geral / ideia | Use as [Discussions](../../discussions) |
| 🔐 Falha de segurança | Siga o [`SECURITY.md`](./SECURITY.md) — **não** abra issue pública |
| 💼 Assunto comercial / parceria | contato@quebradanetwork.com.br |

## 🌐 Links úteis

## ⚠️ Antes de abrir uma issue

1. Verifique se já existe uma issue sobre o mesmo assunto.
2. Confirme que está usando a versão mais recente do projeto.
3. Reúna logs e passos de reprodução — isso acelera bastante a resolução.
