# Assets do perfil

Coloque aqui os arquivos de imagem da organização e descomente as linhas
correspondentes no topo de `profile/README.md`.

| Arquivo | Dimensão sugerida | Uso |
|---------|-------------------|-----|
| `banner.png` | 1920×640 | Banner horizontal do topo do perfil |
| `logo.png` | ~512×512 | Logo principal |
| `logo-mono.png` | ~512×512 | Versão monocromática (opcional) |

> Esta pasta existe só para reservar o caminho `profile/assets/`.
> Pode remover este arquivo depois de adicionar as imagens reais.
